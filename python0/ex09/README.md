# ft_package

A simple sample Python package developed for the Python for Data Science Piscine. This package demonstrates the basic structure required for python packaging and distribution.

## Features

- **count_in_list**: A utility function to count occurrences of a specific item within a list.

## Installation

You can install the package locally using `pip`. Navigate to the root directory where `pyproject.toml` is located and run:

```bash
pip install .
```
